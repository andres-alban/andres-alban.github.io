Some guidelines for understanding the naming convention of the plots for the MARS application

RI = Random Instance

DiscCov = Discrete Covariates

ContCov = Continuous Covariates

1=known labeling; 9=allprog; 15 = allpred; dynamic = dynamic labeling

Example: RI_DiscCov_9 refers to the random instance setting with Discrete Covariates and the allprog labeling
